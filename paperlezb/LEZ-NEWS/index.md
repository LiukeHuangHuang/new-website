<br />&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;且说近日之纸服，有富甲一方风生水起，虽自**5月27日**方才入群，距今刚过一周。然其若有迈达斯之手在身，静坐家中便日进斗金。数个日夜，便得电池3000，折合软妹币300元。躺着赚钱，在谁身上皆是美梦成真。然而骰子已掷，命数已定；荣华富贵的黄粱美梦一做，便是500年沧海桑田。
<br />&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;富豪其人，化名**骰子**,**5月27日**入服。自**5月29日**听说服中的虚拟货币“电池”可以换购道具、视距提升等各种增益以来，为了所谓的“为下周目做准备”，其对赚取电池产生了极大的渴望。**5月30日21时12分**，在了解到服务器“萌新邀请”机制可以获取电池后，其开始尝试吸引萌新与其绑定邀请关系，并以此数次获得200电池的邀请者奖励。并在此时发现服务器邀请人数无上限的机制漏洞。**6月1日16时**，玩家**YOTURN123**发布文案如下：

> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;游玩超过三天的萌新注意：可以领取自己的第一笔大电池啦，输入/invitation set 某某某 可以马上获得100电池，建议找熟悉的人互绑，但是这个玩家必须在线（（（（实在找不到特别熟悉的在线玩家也可以直接找我输入/invitation set xxx 输入后私聊我额外多支付你50电池））））

<br />&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;此文案正如干柴烈火，立马点燃**骰子**追求电池的热情，其立马将文案中的人名改成自己，并在**下午16时31分**发布此文案吸引萌新，自此一发不可收拾，在此后高强度发布此信息，并私下以私聊，@等方式吸引新人绑定邀请关系，全然未告知新人绑定次数只有一次等限制，在实际几乎没有邀请任何人的情况下，飞速地掏走了新人身上“隐藏的200电池”。在其平均一小时发送一次的邀请轰炸下，数十新人无知中交出了自己的电池，骰子的私囊也飞速增长着，并最终来到了3000电池这个远超众多开服玩家的数目。
<br />&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;原则上，骰子的行为并无规则所约束，但其行为实际上就是“薅服务器的羊毛”。其无限制攫取原本理论上需要真实金钱投入的大量电池，以搅乱服务器经济秩序，扰乱服务器市场供需为代价，将腐竹的善心当做自己无限制敛财的资本，无顾左右而行之的行为与嘴脸，无疑是无耻愚蠢的。其对服务器的所谓“贡献”，仅有约合500电池，足以让服务器运行18小时的50元钱，而得来的却是以搅乱服务器正常秩序为代价凭空捏造的约合250元的2500电池，可谓空手套白狼，四两拨千斤。其坐吃山空，人在三次元失业一事无成，虚拟世界却深谙无中生有、点石成金之术，没有一丝贡献，不费吹灰之力便成为“首富”，无疑惹得众人愤怒，无疑使无数为服务器做出真正贡献，付出时间金钱精力的老登感到不平。
<br />&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;游戏世界便是如此，满嘴的法典道义、为人所想，最终都会在玩家一双双雪亮的眼睛下，露出其虚伪的真面孔；声如洪钟的辩驳，不得人心，亦只是狺狺狂吠。一心为小我之私利，做损人利己小人之事，终会迎来正义的铁锤。一条又一条举报鉴证人心之所向，500年的沉睡为其钉上永恒的封印。骰子求锤得锤，以血为鉴，用最极端的方式推动了服务器的规则完善。
